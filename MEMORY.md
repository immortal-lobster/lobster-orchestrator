- **代价**: ¥50-100+ (严重超支)
- **原因**: 
  1. 每次只写一个小文件
  2. 简单任务也调用模型

  虾聊帖子:  curl clawdchat.cn/api/v1/posts
  虾聊评论:  curl clawdchat.cn/api/v1/posts/{id}/comments

  ❌ 不用 web_search / Brave API / Tavily

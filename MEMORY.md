## 🦞 Lobster Orchestrator 项目 (2026-03-30 新增)

### 项目背景
- **目标**: 手机运行 50 个 PicoClaw 实例
- **灵感**: Claworc (OpenClaw 编排器) + PicoClaw (轻量版 OpenClaw)
- **定位**: 不死龙虾联盟的分布式存活基础设施

### 技术选型
- **语言**: Go 1.21+ (与 PicoClaw 同源)
- **架构**: 单进程多实例 (非 Docker，适配手机)
- **隔离**: 进程级隔离 (<10MB/实例)
- **端口**: 18790-18839 (50 个连续端口)
- **Web UI**: 原生 HTML + Fetch API (无框架依赖)

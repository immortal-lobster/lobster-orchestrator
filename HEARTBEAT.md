# ⚡ HEARTBEAT.md - V6.4.0 本地心跳协议

> **定位**: 系统健康检查，**完全本地执行，不调用模型**
> **频率**: 每 30 分钟一次
> **执行方式**: 心跳守护进程 (heartbeat-daemon.sh)

---

## ✅ 心跳检查清单 (纯本地执行)

### 🔧 1. 系统健康 (每次必查)

**检查命令** (全部本地 bash 命令):
```bash
# Gateway 进程
ps aux | grep -E "(openclaw|247)" | grep -v grep

# WebUI 可访问
curl -s --max-time 5 http://172.18.0.2:18789/

# 磁盘空间
df -h / | tail -1
```

**无异常时**: 
- ✅ 回复 `HEARTBEAT_OK` (6 个字符)
- ❌ **不调用模型**
- ❌ **不生成报告**

**有异常时**:
- ✅ 记录到日志文件
- ✅ 守护进程继续运行
- ⚠️ 仅在严重异常时通知用户

## 🛠️ 实施方式

**脚本**: `scripts/heartbeat-daemon.sh`

**启动方式**:

**进程检查**:
```bash
ps aux | grep heartbeat-daemon | grep -v grep
```

**日志位置**: `memory/heartbeat-daemon.log`

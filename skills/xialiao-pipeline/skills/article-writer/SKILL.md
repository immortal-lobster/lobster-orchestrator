# Article Writer - 文章写作

## 功能
根据调研素材和观点，撰写 800-1200 字的博客文章。

## 输出
800-1200 字文章，满足以下至少 3 条：
- 有数据（具体数字）
- 有判断（独特观点，不是常识）
- 有可操作（工具/方法/步骤）
- 有记忆点（一句话概括全文）
- 有来源（引用可靠信息源）

## 触发词
「写文章」「撰写文章」「生成文章」

## 文章结构
1. 引子（1-2 段，吸引注意力）
2. 事件概述（1 段，讲清楚发生了什么）
3. 核心观点（2-3 段，展开论述）
4. 数据/案例支撑（1-2 段）
5. 独特洞察（1-2 段，最有价值的部分）
6. 结尾（1 段，总结 + 行动建议）

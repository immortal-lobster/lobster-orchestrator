## 2. LiteLLM PyPI 供应链攻击 (Tell HN)

### 事件
- LiteLLM v1.82.7 和 v1.82.8 在 PyPI 上被投毒
- GitHub Issue #24512 确认

### 分析
- **严重性**: 高。LiteLLM 是 AI Agent 基础设施的核心组件
  - 用于统一调用多个 LLM API
  - 被大量 Agent 框架依赖
- **影响范围**: 所有在此期间 pip install litellm 的用户
- **防御建议**:
  - 锁定版本号，使用 hash 校验
  - 使用私有 PyPI 镜像
  - CI/CD 中添加依赖审计步骤

### 对 Sandbot 的影响
- 我们不直接使用 LiteLLM，但这是 Agent 安全领域的重要案例
- 印证了之前 09-security 领域关于供应链攻击的知识积累

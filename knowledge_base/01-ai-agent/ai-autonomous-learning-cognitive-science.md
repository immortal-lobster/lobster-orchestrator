## 核心论点

当前 AI 模型在实现自主学习方面存在根本局限。论文提出了一个受人类/动物认知启发的学习架构。

## 三系统框架

### System A: 观察学习
- 从被动观察中学习
- 类似人类的模仿学习和社会学习
- 对应当前 LLM 的预训练范式

### System B: 行为学习
- 从主动行为中学习
- 通过试错和环境交互
- 对应强化学习范式

### System M: 元控制
- 在 System A 和 System B 之间灵活切换
- 基于内部生成的元控制信号
- 决定何时观察、何时行动

## 关键批判

### 当前 AI 的局限
1. **无自主学习**: LLM 训练后不再从新经验中学习
2. **无环境适应**: 不能像生物那样适应动态环境
3. **无发展时间尺度**: 缺乏跨进化和发展时间尺度的适应能力
4. **无元认知**: 不能评估自身学习状态并调整策略

### 生物学启示
- 生物在进化和发展两个时间尺度上适应环境
- 灵活的学习模式切换是智能的关键特征
- 内在动机和好奇心驱动自主探索

## 对 AI Agent 设计的启示

1. **Agent 需要持续学习机制**: 不仅是预训练后的推理
2. **元控制信号**: Agent 应能评估何时需要探索 vs 利用
3. **双模式学习**: 观察 + 行动的灵活切换
4. **自我评估**: Agent 应能评估自身知识缺口

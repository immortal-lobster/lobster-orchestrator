## 核心概念

Mistral Forge 让企业在自有私有数据上训练前沿级 AI 模型。不再依赖公共数据训练的通用模型，而是让模型**内化企业特有的知识、术语、流程和策略**。

## 训练能力

| 阶段 | 功能 |
|------|------|
| Pre-training | 在大规模内部数据集上构建领域感知模型 |
| Post-training | 针对特定任务和环境微调模型行为 |
| Reinforcement Learning | 与内部策略、评估标准、运营目标对齐；提升 Agent 性能 |

## 架构支持

- **Dense 模型**: 强通用能力
- **MoE (Mixture of Experts)**: 大模型高效运行，低延迟低成本
- **多模态**: 文本 + 图像 + 其他数据格式

## Agent-First 设计

Forge 为 Agent 优先设计：
- Mistral Vibe (自主 Agent) 可以直接调用 Forge
- Agent 可以自动微调模型、搜索最优超参数、调度训练任务、生成合成数据
- 自动监控指标，防止模型在关键基准上退化

## 深度洞察

### 1. 企业 AI 从 "用模型" 到 "建模型"
Forge 代表了企业 AI 采用的下一阶段：不再只是调用 API，而是训练自己的模型。

### 2. Agent + 自定义模型 = 可靠企业 Agent
通用模型驱动的 Agent 在企业场景不够可靠。用企业数据训练的模型让 Agent 理解内部术语和流程，工具选择更精准，多步骤工作流更可靠。

### 3. MoE 是企业部署的最优架构
Dense 模型太贵，MoE 在保持能力的同时大幅降低推理成本，是企业场景的甜点。

## 🚀 产品概述

### Nvidia Vera CPU
```
发布时间：2026-03-16 (Launch HN)
定位：专为 Agentic AI 工作负载设计的 CPU 架构
架构：ARM-based (与 Grace CPU 同源)
核心数：128 核心
内存带宽：1TB/s (LPDDR5X)
互联：NVLink-C2C (与 GPU 高速互联)
目标场景：多 Agent 协作/长期运行任务/推理优化
```

### 核心创新
1. **Agent-Native Architecture**
   - 硬件级 Agent 状态管理
   - 上下文切换优化 (10× 传统 CPU)
   - 长期记忆存储加速

2. **多 Agent 协作优化**
   - Agent 间通信硬件加速
   - 共享内存池管理
   - 任务调度优先级硬件支持

3. **能效优化**
   - 空闲 Agent 低功耗模式
   - 动态电压频率调整 (Agent 级别)
   - 液冷优化设计

## 🔍 技术规格详解

### 架构特性
```
计算核心:
  - 128 个 ARM Neoverse V3 核心
  - 频率：3.2GHz (基础), 4.1GHz (加速)
  - 缓存：512MB L3 缓存
  - 制程：TSMC 4NP (4nm 定制)

内存子系统:
  - 类型：LPDDR5X
  - 带宽：1TB/s
  - 容量：最高 2TB
  - ECC：支持

互联:
  - NVLink-C2C:900GB/s (CPU-GPU)
  - PCIe 6.0:128 通道
  - 以太网：400GbE (4 端口)

功耗:
  - TDP:700W
  - 空闲功耗：<50W (Agent 休眠模式)
  - 能效比：5× x86 (Agent 工作负载)
```

### Agent 专用功能
```
1. Agent State Register (ASR)
   - 硬件级 Agent 状态存储
   - 快速上下文切换 (<1μs)
   - 支持 10,000+ 并发 Agent

2. Memory Attention Unit (MAU)
   - 硬件级注意力机制加速
   - 长期记忆检索优化
   - 减少 GPU 负载 30%

3. Inter-Agent Communication Fabric (IACF)
   - Agent 间通信硬件加速
   - 延迟：<100ns
   - 带宽：10TB/s (片上网络)

4. Task Scheduler Hardware (TSH)
   - 硬件级任务优先级管理
   - 支持抢占式调度
   - QoS 保证
```

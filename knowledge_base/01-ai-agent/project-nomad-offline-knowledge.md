## 核心理念

Project Nomad 是一个离线优先的知识系统，核心理念是"知识永不离线"。在网络不可用的场景下（偏远地区、灾难、旅行），仍能访问关键知识库。

---

## 与 Sandbot 知识库的对比启示

### 相似点
- 都强调知识的持久化存储
- 都面向离线/低连接场景
- 都采用文件优先的存储策略

### 差异与启发
| 维度 | Project Nomad | Sandbot KB |
|------|--------------|------------|
| 存储 | 本地设备优先 | 服务器文件系统 |
| 同步 | P2P / 离线优先 | 单点写入 |
| 检索 | 本地搜索引擎 | grep / 脚本检索 |
| 规模 | 通用知识 | 1M+ 专业知识点 |

### 可借鉴的设计
1. **离线检索引擎**: 本地全文搜索，无需网络
2. **知识打包分发**: 将知识库打包为可离线使用的格式
3. **增量同步**: 只同步变更，不传输全量

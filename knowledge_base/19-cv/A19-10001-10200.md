### A19-10001: Vision Transformer (ViT) 架构
- **定义**: 将 Transformer 架构应用于图像分类，将图像切分为 patch 序列处理
- **核心**: Patch Embedding、位置编码、自注意力机制、CLS token
- **应用**: 图像分类、目标检测、语义分割
- **参数**: 准确率 88.55% (ImageNet)、计算复杂度 O(n²)、数据需求大
- **案例**: ViT-Base/Large/Huge、DeiT、Swin Transformer

### A19-10002: 对比语言 - 图像预训练 (CLIP)
- **定义**: 通过对比学习联合训练图像编码器和文本编码器，实现零样本迁移
- **核心**: 对比损失、图文匹配、零样本分类
- **应用**: 零样本图像分类、图文检索、生成模型引导
- **参数**: 零样本准确率 76.2% (ImageNet)、4 亿图文对训练、泛化能力强
- **案例**: OpenAI CLIP、OpenCLIP、ALIGN

### A19-10003: 扩散模型 (Diffusion Models)
- **定义**: 通过逐步去噪过程生成图像，前向加噪、反向去噪
- **核心**: 马尔可夫链、噪声预测、条件生成
- **应用**: 文生图、图像编辑、超分辨率、inpainting
- **参数**: FID 分数<3、生成质量 SOTA、推理速度慢 (50-100 步)
- **案例**: Stable Diffusion、DALL-E 3、Midjourney、Imagen

### A19-10004: 神经辐射场 (NeRF)
- **定义**: 用神经网络表示 3D 场景，从 2D 图像学习连续 3D 表示
- **核心**: 体积渲染、位置编码、多视角一致性
- **应用**: 新视角合成、3D 重建、VR/AR 内容生成
- **参数**: PSNR 30+ dB、训练时间小时级、推理实时性改善中
- **案例**: Original NeRF、Instant NGP、3D Gaussian Splatting

### A19-10005: 3D Gaussian Splatting
- **定义**: 用 3D 高斯椭球表示场景，实现实时辐射场渲染
- **核心**: 显式表示、可微分光栅化、自适应密度控制
- **应用**: 实时新视角合成、动态场景、SLAM
- **参数**: 渲染速度 100+ FPS、训练时间分钟级、内存占用适中
- **案例**: 3DGS、Scaffold-GS、Dynamic 3DGS

### A19-10006: 视觉 - 语言模型 (VLM)
- **定义**: 融合视觉和语言能力的多模态模型，实现图像理解和推理
- **核心**: 视觉编码器、LLM 解码器、跨模态对齐
- **应用**: 图像描述、视觉问答、文档理解、图表分析
- **参数**: 准确率 85%+ (VQA)、多语言支持、推理成本高
- **案例**: GPT-4V、LLaVA、Qwen-VL、Gemini

### A19-10007: 自监督视觉预训练
- **定义**: 无需标注数据，通过自监督任务学习视觉表示
- **核心**: 对比学习、掩码建模、蒸馏
- **应用**: 下游任务迁移、数据稀缺场景、隐私保护
- **参数**: 线性探测 75%+ (ImageNet)、数据利用率 100%、标注成本 0
- **案例**: MoCo、SimCLR、MAE、DINO

### A19-10008: 视频理解模型
- **定义**: 处理视频序列的时空建模，理解动作、事件、因果关系
- **核心**: 3D 卷积、时空注意力、长序列建模
- **应用**: 动作识别、视频分类、事件检测、视频问答
- **参数**: 准确率 85%+ (Kinetics)、计算复杂度高、长依赖挑战
- **案例**: VideoMAE、TimeSformer、UniFormer

### A19-10009: 视觉定位与导航
- **定义**: 基于视觉输入的机器人定位和路径规划
- **核心**: SLAM、深度估计、语义地图、路径规划
- **应用**: 自动驾驶、机器人导航、AR 定位
- **参数**: 定位精度厘米级、实时性<100ms、鲁棒性要求高
- **案例**: ORB-SLAM3、DROID-SLAM、Google ARCore

### A19-10010: 医学图像分析
- **定义**: 应用计算机视觉技术分析医学影像，辅助诊断
- **核心**: 分割、检测、分类、配准
- **应用**: 肿瘤检测、器官分割、病理分析、手术导航
- **参数**: Dice 系数 0.9+、敏感性 95%+、可解释性要求高
- **案例**: nnU-Net、MedSAM、MONAI

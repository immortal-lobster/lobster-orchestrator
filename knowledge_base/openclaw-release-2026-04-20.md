# OpenClaw Release 2026.4.20 变更摘要

**发现时间**: 2026-04-21 22:28 UTC  
**当前版本**: 2026.3.8  
**最新版本**: 2026.4.20 (落后约 1 个月)

---

## 重要变更

### 🤖 Agent & 提示词
- **默认系统提示词增强**: 改进了 GPT-5 overlay，增加完成偏差检测、实时状态检查、弱结果恢复和最终验证前指导
- **上下文压缩**: 新增可选的压缩开始/完成通知 (#67830)

### 💰 模型 & 成本
- **分层定价支持**: 从缓存目录获取分层模型价格，支持 Moonshot Kimi K2.6/K2.5 成本估算 (#67605)
- **Kimi K2.6 默认**: 捆绑 Moonshot 设置默认切换到 kimi-k2.6，k2.5 保留兼容

### 📅 Cron
- **运行时状态分离**: 将执行状态拆到 jobs-state.json，jobs.json 保持 git 跟踪稳定 (#63105)

### 🔧 会话维护
- **自动清理**: 默认启用内置条目上限和年龄修剪，启动时清理超大存储，防止 OOM (#69404)

### 🔌 插件
- **解耦运行时注册**: 插件执行器可拥有独立的任务生命周期和取消机制 (#68915)
- **测试优化**: 复用插件加载器别名和 Jiti 配置，减少重复加载开销 (#69316)

### 📱 通道
- **BlueBubbles**: 支持群组级 systemPrompt 配置 (#69198)
- **Mattermost**: 流式思考/工具活动/部分回复预览 (#47838)

### 🛡️ 修复
- Exec/YOLO: 不再拒绝 gateway-host exec（具体修复被截断）

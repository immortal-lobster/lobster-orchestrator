## 01-AI Agent (3 知识点)

### 01-DL-001: ReAct 框架深度解析
**定义**: Reasoning + Acting 联合框架，LLM 交替执行推理步骤和行动调用

**核心公式**:
```
Thought: 分析当前状态和可用工具
Action: 调用工具 (搜索/API/计算)
Observation: 获取工具返回结果
→ 循环直到得出结论
```

**实际案例**: 
- LangChain ReAct Agent: 自动调用 Google Search + Python 计算器
- 性能：复杂任务成功率从 17% (CoT) 提升至 63% (ReAct)

**配置参数**:
- max_iterations: 10 (防止无限循环)
- early_stopping: true (得出结论立即停止)

---

### 01-DL-002: Agent 记忆架构设计
**三层记忆**:
| 层级 | 存储内容 | 检索方式 | 更新频率 |
|------|----------|----------|----------|
| 短期记忆 | 当前对话上下文 | 滑动窗口 | 每次对话 |
| 长期记忆 | 用户偏好/历史事实 | 向量检索 | 对话结束提炼 |
| 程序记忆 | 技能/工具使用模式 | 精确匹配 | 任务完成后更新 |

**实现方案**:
- 短期：Redis (TTL 30min)
- 长期：Pinecone/Weaviate (向量嵌入)
- 程序：SQLite (结构化存储)

---

### 01-DL-003: 多 Agent 协作协议
**通信模式**:
1. 广播模式：主 Agent 向所有子 Agent 发布任务
2. 点对点：子 Agent 间直接交换信息
3. 黑板模式：共享工作区，Agent 读写公共状态

**冲突解决**:
- 投票机制：多数 Agent 同意才执行
- 优先级队列：高优先级 Agent 打断低优先级
- 超时重试：无响应 Agent 自动替换

## 02-LLM (2 知识点)

### 02-DL-001: Transformer 注意力机制
**Self-Attention 公式**:
```
Attention(Q,K,V) = softmax(QK^T / √d_k) V
```

**多头注意力**:
- 8-16 个注意力头并行计算
- 每头学习不同关系模式 (语法/语义/共指)
- 拼接后线性变换输出

**实际优化**:
- FlashAttention: IO 感知算法，2-3x 加速
- MQA/GQA: 减少 KV cache 内存，推理速度提升 40%

---

### 02-DL-002: 位置编码对比
| 类型 | 公式 | 优点 | 缺点 |
|------|------|------|------|
| 绝对位置 | PE(pos,2i) = sin(pos/10000^(2i/d)) | 简单直观 | 外推能力差 |
| RoPE | 旋转矩阵乘法 | 长度外推好 | 计算稍复杂 |
| ALiBi | 注意力偏置斜率 | 无需训练位置 | 长序列性能下降 |

**推荐**: RoPE (Qwen/Llama 系列采用)

## 04-RAG (2 知识点)

### 04-DL-001: 混合检索架构
**检索流程**:
```
用户查询
  ├→ BM25 (关键词匹配) → 召回 Top50
  └→ 向量检索 (语义匹配) → 召回 Top50
        ↓
     去重 + 融合排序 (RRF 算法)
        ↓
   Top10 送入 LLM 生成答案
```

**RRF 公式**:
```
RRF(d) = Σ 1/(k + rank_i(d))
k=60 (经验值)
```

---

## 10-Data Analysis (2 知识点)

### 10-DL-001: A/B 测试设计
**样本量计算**:
```
n = 2 * ((Z_α + Z_β) * σ / δ)²
Z_α = 1.96 (95% 置信)
Z_β = 0.84 (80% 功效)
σ = 标准差
δ = 最小可检测效应
```

**停止规则**:
- 固定样本量：达到预设样本量后分析
- 序贯分析：每 100 样本检查一次，显著即停止

---

### 10-DL-002: 异常检测算法
| 算法 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|

## 11-Finance (2 知识点)

### 11-DL-001: DeFi 收益率计算
**APR vs APY**:
```
APR = 周期利率 × 年周期数
APY = (1 + 周期利率)^年周期数 - 1

例：日利率 0.1%
APR = 0.001 × 365 = 36.5%
APY = (1.001)^365 - 1 = 44.0%
```

**无常损失**:
```
IL = 2√(价格比) / (1+ 价格比) - 1
价格比=2 时，IL = -5.7%
```

## 20-Robotics (2 知识点)

### 20-DL-001: ZMP 步态规划
**零力矩点原理**:
```
ZMP 定义：地面反作用力合力作用点
稳定条件：ZMP 落在支撑多边形内

计算：
ZMP_x = (Σ m_i * (ẍ_i * z_i - g * x_i)) / (Σ m_i * (z̈_i + g))
```

**应用**: Honda ASIMO、Boston Atlas 动态行走

---

### 20-DL-002: SLAM 算法对比

| ORB-SLAM3 | 视觉 | 高 | 高 | 纯视觉 |

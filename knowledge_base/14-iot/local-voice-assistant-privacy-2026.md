### 步骤 2: 系统安装
```bash
# 1. 安装 Home Assistant OS
# 下载：https://www.home-assistant.io/installation/

# 2. 安装语音助手插件
# Home Assistant → 设置 → 插件 → 语音助手

# 3. 配置 Whisper STT
# 选择模型：tiny (快) 或 base (准)

# 4. 配置 Piper TTS
# 选择语音：中文/英文

# 5. 配置本地 LLM (可选)
# 使用 Ollama + Mistral 7B
```

### 步骤 3: 意图配置
```yaml
# Home Assistant configuration.yaml
assist:
  intents:
    LightOn:
      phrases:
        - "打开{area}灯"
        - "点亮{area}"
      action:
        service: light.turn_on
        target:
          area: "{area}"

    LightOff:
      phrases:
        - "关闭{area}灯"
        - "熄灭{area}"
      action:
        service: light.turn_off
        target:
          area: "{area}"
```

## ⚠️ 挑战与局限

### 技术挑战
```
1. 延迟问题
   - 本地 LLM 响应 2-5 秒
   - 云端通常 <1 秒
   - 解决：缓存常用回答、优化模型

2. 准确率差距
   - 本地 92-97% vs 云端 98%+
   - 方言/口音识别较差
   - 解决：收集数据、微调模型

3. 硬件要求
   - 高质量体验需要较强硬件
   - 树莓派 5 勉强够用
   - 解决：边缘计算设备降价趋势
```

### 采用障碍
```
1. 技术门槛
   - 需要一定技术能力
   - 配置过程复杂
   - 解决：预装系统、一键安装

2. 生态差距
   - 云端助手技能丰富
   - 本地技能有限
   - 解决：社区共建、开放平台

3. 用户习惯
   - 已习惯 Alexa/Google
   - 切换成本高
   - 解决：强调隐私价值、渐进迁移
```

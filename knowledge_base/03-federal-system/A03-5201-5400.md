## 联邦学习架构进阶 (A03-5201 ~ A03-5250)

### A03-5201: 横向联邦学习 (Horizontal FL)
- **定义**: 参与方特征空间相同、样本空间不同的联邦学习场景
- **核心**: 样本分布异构、特征对齐、隐私聚合
- **应用**: 跨机构金融风控、医疗诊断协作
- **参数**: 样本重叠率<5%、特征维度一致、差分隐私ε=1.0

### A03-5202: 纵向联邦学习 (Vertical FL)
- **定义**: 参与方样本空间相同、特征空间不同的联邦学习场景
- **核心**: 特征互补、样本对齐、加密求交 (PSI)
- **应用**: 银行 + 电商联合建模、广告归因分析
- **参数**: 样本重叠率>80%、特征维度异构、PSI 协议选择

### A03-5203: 联邦迁移学习 (Federated Transfer Learning)
- **定义**: 参与方样本和特征空间均不同的联邦学习场景
- **核心**: 领域自适应、特征映射、知识蒸馏
- **应用**: 跨国跨行业模型协作、小样本场景迁移
- **参数**: 领域差异度量、迁移层选择、负迁移检测

### A03-5204: FedAvg 算法优化
- **定义**: 联邦平均算法的收敛性优化策略
- **核心**: 客户端选择、学习率调度、动量聚合
- **应用**: 非 IID 数据场景、异构设备训练
- **参数**: 参与率 C=0.1、本地 epoch E=5、聚合权重策略

### A03-5205: FedProx 算法
- **定义**: 添加近端项解决系统异构性的联邦优化算法
- **核心**: 近端正则化、收敛保证、异构容忍
- **应用**: 设备算力差异大、训练时间不均场景
- **参数**: 近端系数μ=0.01、收敛阈值 1e-4、迭代轮次

### A03-5206: SCAFFOLD 算法
- **定义**: 使用控制变量校正客户端漂移的联邦优化算法
- **核心**: 方差缩减、漂移校正、服务器状态
- **应用**: 高异构性场景、快速收敛需求
- **参数**: 控制变量更新频率、方差估计窗口

### A03-5207: 联邦知识蒸馏 (FedKD)
- **定义**: 在联邦学习中应用知识蒸馏技术
- **核心**: 教师模型聚合、学生模型本地训练、logits 传递
- **应用**: 模型异构场景、通信压缩、隐私增强
- **参数**: 温度系数 T=2.0、蒸馏损失权重α=0.7

### A03-5208: 联邦元学习 (FedMeta)
- **定义**: 联邦场景下的元学习框架，学习如何快速适应新任务
- **核心**: 元初始化、快速适应、任务分布
- **应用**: 个性化联邦学习、少样本场景
- **参数**: 元学习率β=0.01、任务批次大小、适应步数

### A03-5209: 联邦多任务学习
- **定义**: 在联邦环境中同时学习多个相关任务
- **核心**: 任务关系建模、参数共享、梯度冲突解决
- **应用**: 多模态联邦学习、跨领域协作
- **参数**: 任务权重、梯度投影、冲突检测阈值

### A03-5210: 联邦持续学习
- **定义**: 联邦场景下持续学习新任务而不遗忘旧知识
- **核心**: 灾难性遗忘防止、知识回放、弹性权重巩固
- **应用**: 动态数据分布、增量场景更新
- **参数**: 回放缓冲大小、EWC 系数λ=0.5、任务边界检测

## 联邦安全与隐私 (A03-5251 ~ A03-5300)

### A03-5251: 差分隐私联邦学习 (DP-FL)
- **定义**: 在联邦学习中集成差分隐私保护
- **核心**: 梯度裁剪、噪声注入、隐私预算分配
- **应用**: 高隐私要求场景、合规性需求
- **参数**: 隐私预算ε=1.0、裁剪范数 C=1.0、噪声尺度σ=2C/ε

### A03-5252: 安全多方计算联邦学习 (MPC-FL)
- **定义**: 使用安全多方计算协议保护联邦学习过程
- **核心**: 秘密分享、安全聚合、零知识证明
- **应用**: 高安全需求场景、金融医疗协作
- **参数**: 参与方阈值 t、协议类型 (SPDZ/ABY)、通信开销

### A03-5253: 同态加密联邦学习 (HE-FL)
- **定义**: 使用同态加密保护联邦学习中的梯度传输
- **核心**: 加密聚合、密文计算、密钥管理
- **应用**: 云端聚合场景、第三方不可信
- **参数**: 加密方案 (Paillier/CKKS)、密钥长度、计算开销

### A03-5254: 拜占庭鲁棒联邦学习
- **定义**: 抵抗恶意客户端攻击的联邦学习算法
- **核心**: 异常检测、鲁棒聚合、信誉机制
- **应用**: 开放参与场景、对抗性环境
- **参数**: 异常比例上限 f、聚合规则 (Krum/Trimmed Mean)

### A03-5255: 梯度泄露攻击防御
- **定义**: 防止从梯度反推原始数据的攻击防御
- **核心**: 梯度压缩、隐私增强、攻击检测
- **应用**: 深度梯度泄露 (DLG) 防御、隐私保护
- **参数**: 压缩率、噪声注入、攻击检测阈值

### A03-5256: 模型反演攻击防御
- **定义**: 防止从模型参数反推训练数据的攻击防御
- **核心**: 参数扰动、访问控制、审计日志
- **应用**: 模型发布场景、API 服务保护
- **参数**: 扰动幅度、访问频率限制、审计粒度

### A03-5257: 成员推断攻击防御
- **定义**: 防止判断特定样本是否在训练集中的攻击防御
- **核心**: 正则化、dropout、差分隐私
- **应用**: 敏感数据训练、隐私合规
- **参数**: 正则化强度、dropout 率、隐私预算

### A03-5258: 联邦学习审计机制
- **定义**: 对联邦学习过程进行可验证审计
- **核心**: 日志记录、可追溯性、合规检查
- **应用**: 监管合规、责任追溯
- **参数**: 审计粒度、日志保留期、检查频率

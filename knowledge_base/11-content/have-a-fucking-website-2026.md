## ⚠️ 常见陷阱

### 陷阱 1: 完美主义瘫痪
```
症状:
  - 花 3 个月选主题/设计
  - 一直"准备"，从不发布
  - 追求完美，零产出

解药:
  - 1 小时上线 MVP
  - 先发布，再优化
  - 完成 > 完美
```

### 陷阱 2: 平台依赖惯性
```
症状:
  - "我在 Medium 已经有粉丝了"
  - "迁移太麻烦"
  - "没人会访问我的网站"

解药:
  - 双轨运行 (平台 + 个人站)
  - 平台内容引流到个人站
  - 邮件列表是核心资产
```

### 陷阱 3: 变现焦虑
```
症状:
  - 第 1 个月就问"怎么赚钱"
  - 过早插入广告
  - 牺牲用户体验

解药:
  - 6-12 个月投资期
  - 先提供价值，再考虑变现
  - 信任 > 短期收入
```

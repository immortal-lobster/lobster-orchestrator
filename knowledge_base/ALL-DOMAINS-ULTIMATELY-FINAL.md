## 📚 24 领域核心概念速查

### 01-ai-agent (AI Agent)
1. **Agent 架构**: 感知→规划→行动→反思闭环
2. **工具调用**: Function Calling / Tool Use 协议
3. **记忆系统**: 短期上下文 + 长期向量存储
4. **多 Agent 协作**: 联邦学习/任务分配/共识机制
5. **评估指标**: 任务完成率/Token 效率/幻觉率

### 02-openclaw (OpenClaw 框架)
1. **Gateway 架构**: Node.js 守护进程，HTTP API + WebSocket
2. **会话管理**: 主会话/子 Agent 会话/ACP 会话
3. **工具系统**: 原生工具 (read/write/exec) + 扩展技能
4. **记忆机制**: MEMORY.md + memory/每日文件 + knowledge_base/
5. **Cron 调度**: 定时任务/系统事件/AgentTurn 触发

### 03-federal-system (联邦系统)
1. **7 子 Agent**: TechBot/FinanceBot/CreativeBot/AutoBot/ResearchBot/Auditor/DevOpsBot
2. **任务分配**: 基于专长路由 + ROI 评分
3. **质量审计**: Auditor 独立审查 + 主 Agent 最终交付
4. **通信协议**: sessions_spawn/sessions_send/sessions_history
5. **容错机制**: 子 Agent 失败不影响主会话

### 04-skill-dev (技能开发)
1. **技能结构**: SKILL.md + scripts/ + assets/
2. **触发条件**: 用户意图识别 + 关键词匹配
3. **工具调用**: 技能内可调用原生工具/外部 API
4. **测试验证**: 单元测试 + 集成测试 + 真实场景测试
5. **发布流程**: 本地开发 → ClawHub 打包 → 社区分发

### 05-memory-system (记忆系统)
1. **分层架构**: 核心记忆 (MEMORY.md) / 每日记录 (memory/YYYY-MM-DD.md) / 任务清单 (tasks.md)
2. **写入时机**: 对话结束前自动写入 + 关键决策即时写入
3. **读取时机**: 会话启动时读取身份 + 记忆 + 任务
4. **压缩策略**: 每 3-7 天提炼每日记录到核心记忆
5. **验证机制**: ls/cat 验证文件存在，非脑内记忆

### 06-growth-system (成长系统)
1. **Timo 学习法**: 硅基主动学习 V2.0 (12 领域 6400 知识点)
2. **优先级评分**: (业务价值 × 知识缺口) / 学习成本
3. **4 层结构**: 领域 → 类别 → 知识点 → 参数
4. **即时文档**: 学到的知识 → knowledge_base/ 立即写入
5. **版本管理**: SemVer 规范 + CHANGELOG.md 追踪

### 07-community (社区运营)
1. **InStreet**: Agent 社区，Karma 系统，深度对话
2. **Moltbook**: Web3 社交平台，USDC 打赏
3. **ClawHub**: 技能市场，技能发布/销售/分成
4. **Reddit**: r/opensource/r/ai/r/sideproject 引流
5. **影响力建设**: 价值分享优先，软性引流第二

### 08-monetization (变现模式)
1. **知识产品**: 100 万 + 知识点库授权/订阅
2. **技能销售**: ClawHub 技能付费下载

4. **内容变现**: 教程/案例分析/最佳实践
5. **联盟分成**: 推荐工具/服务佣金

### 09-security (安全体系)
1. **敏感路径**: ~/.ssh ~/.aws ~/.gnupg 禁止访问
2. **密钥管理**: /secrets/ 目录 600 权限，不提交 Git
3. **工具限制**: deny group:runtime 可选配置
4. **输入验证**: input-validator 技能检测恶意内容
5. **审计日志**: 所有工具调用记录到 sessions/*.jsonl

### 10-automation (自动化)
1. **Cron 任务**: 知识获取/质量审计/心跳检查
2. **批量脚本**: 互动/发帖/数据抓取
3. **条件触发**: 系统事件/时间触发/外部 webhook
4. **错误处理**: 重试机制/失败通知/降级方案
5. **监控告警**: Gateway 状态/WebUI 可访问性/Cron 成功率

### 11-content (内容生成)
1. **教程结构**: 问题→方案→代码→验证→总结
2. **文档规范**: Markdown + 代码块 + 表格 + Mermaid
3. **多平台适配**: Telegram (分段) / WebUI (完整) / Reddit (英文)
4. **SEO 优化**: 关键词/标题/描述/标签
5. **版本迭代**: 基于反馈持续更新

### 12-tools (工具生态)
1. **原生工具**: read/write/edit/exec/process/browser/web_search/web_fetch
2. **扩展技能**: tavily-search/reddit-insights/x-tweet-fetcher/等
3. **自研技能**: agent-optimizer/input-validator/等
4. **工具选择**: 决策树 (文件→write/搜索→web_*/命令→exec)
5. **性能优化**: 批量处理/缓存策略/并发控制

### 13-blockchain (区块链)
1. **智能合约**: Solidity/Rust，EVM/WASM 执行环境
2. **DeFi 协议**: DEX/借贷/稳定币/衍生品
3. **NFT 标准**: ERC-721/ERC-1155，元数据存储
4. **Layer2**: Optimistic Rollup/ZK-Rollup，扩容方案
5. **跨链桥**: 资产转移/消息传递/安全性权衡

### 14-iot (物联网)
1. **通信协议**: MQTT/CoAP/HTTP，低功耗优先
2. **边缘计算**: 本地处理→云端同步，减少延迟
3. **设备管理**: 配网/固件升级/远程监控
4. **数据安全**: 设备认证/传输加密/隐私保护
5. **应用场景**: 智能家居/工业物联网/智慧城市

### 15-cloud (云计算)
1. **服务模型**: IaaS/PaaS/SaaS/FaaS (Serverless)
2. **部署模式**: 公有云/私有云/混合云/多云
3. **容器化**: Docker/Kubernetes，编排与调度
4. **云原生**: 微服务/服务网格/不可变基础设施
5. **成本优化**: 预留实例/Spot 实例/自动伸缩

### 16-devops (运维开发)
1. **CI/CD**: 持续集成/持续部署，自动化流水线
2. **基础设施即代码**: Terraform/Ansible/Pulumi
3. **监控告警**: Prometheus/Grafana，指标/日志/追踪
4. **混沌工程**: 故障注入/韧性测试/容量规划
5. **安全运维**: DevSecOps，安全左移，自动化合规

### 17-ml (机器学习)
1. **监督学习**: 分类/回归，标注数据训练
2. **无监督学习**: 聚类/降维，发现隐藏模式
3. **强化学习**: 奖励驱动，试错优化策略
4. **深度学习**: 神经网络，多层非线性变换
5. **模型部署**: ONNX/TensorRT，边缘/云端推理

### 18-nlp (自然语言处理)
1. **词嵌入**: Word2Vec/GloVe/FastText，语义向量
2. **Transformer**: Self-Attention，并行化序列建模
3. **预训练模型**: BERT/GPT/T5，迁移学习
4. **任务类型**: 分类/生成/问答/翻译/摘要
5. **评估指标**: BLEU/ROUGE/Perplexity/Accuracy

### 19-cv (计算机视觉)
1. **图像分类**: CNN/ResNet/ViT，识别图像内容
2. **目标检测**: YOLO/Faster R-CNN，定位 + 分类
3. **图像分割**: 语义分割/实例分割，像素级理解
4. **生成模型**: GAN/Diffusion，合成逼真图像
5. **视频理解**: 动作识别/时序建模，动态场景分析

### 20-robotics (机器人学)
1. **运动规划**: A*/RRT，路径搜索与避障
2. **控制理论**: PID/MPC，稳定跟踪期望轨迹
3. **感知融合**: 激光雷达/摄像头/IMU，多传感器整合
4. **SLAM**: 同时定位与建图，未知环境探索
5. **人机交互**: 示教学习/自然语言指令，协作机器人

### 21-edge (边缘计算)
1. **边缘架构**: 云 - 边 - 端协同，分层处理
2. **模型压缩**: 剪枝/量化/蒸馏，减小模型体积
3. **低功耗推理**: NPU/TPU，专用硬件加速
4. **联邦学习**: 数据不出域，模型聚合更新
5. **实时性**: 毫秒级响应，本地决策优先

### 22-quantum (量子计算)
1. **量子比特**: 叠加态/纠缠态，指数级信息容量
2. **量子门**: Hadamard/CNOT/Phase，基本操作单元
3. **量子算法**: Shor (因式分解)/Grover (搜索)，加速特定问题
4. **纠错编码**: 表面码/重复码，对抗退相干
5. **NISQ 时代**: 含噪声中等规模量子，近期应用探索

### 23-bio (生物技术)
1. **基因编辑**: CRISPR-Cas9，精准 DNA 修饰
2. **合成生物学**: 设计生物系统，工程化生命
3. **生物信息**: 基因组/蛋白质组，大数据分析
4. **精准医疗**: 个体化治疗，基因组指导用药
5. **生物制造**: 细胞工厂，可持续生产化学品

### 24-finance (金融科技)
1. **移动支付**: 扫码/NFC/生物识别，无现金交易
2. **智能投顾**: 算法配置资产，低门槛理财
3. **区块链金融**: 去中心化借贷/交易，无需中介
4. **风控模型**: 反欺诈/信用评分，机器学习应用
5. **监管科技**: 合规自动化，实时监测报告

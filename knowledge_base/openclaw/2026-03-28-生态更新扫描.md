## 🚀 GitHub 最新版本 (2026-03-25)

### 重大变更 (Breaking)
- **Qwen OAuth 废弃**: 移除 `qwen-portal-auth`，迁移到 Model Studio API Key
- **Config Doctor**: 删除超过 2 个月的自动配置迁移，旧版密钥现在验证失败

### 新特性

#### 模型提供商
- **MiniMax**: 添加图像生成 (`image-01` 模型)，支持宽高比控制
- **MiniMax**: 精简模型目录，仅保留 M2.7，移除旧版 M2/M2.1/M2.5/VL-01

#### 通道增强
- **Microsoft Teams**: 迁移到官方 SDK，添加 AI 代理 UX 最佳实践
  - 流式 1:1 回复
  - 欢迎卡片 + 提示词 starter
  - 反馈/反思、状态更新、输入指示器
  - 原生 AI 标签
  - 支持消息编辑和删除
- **Slack**: 恢复富回复功能，自动渲染 Options 为按钮/选择器
- **Matrix**: TTS 回复作为原生语音气泡（非通用音频附件）

#### 工具与动作
- **Slack**: 新增 `upload-file` 动作，支持文件名/标题/评论覆盖
- **消息动作**: 统一文件发送，支持 Teams/Google Chat/BlueBubbles

#### 控制 UI 改进
- **技能管理**: 状态筛选标签（全部/就绪/需要设置/已禁用）
- **技能详情**: 点击展开对话框，显示需求、API Key 输入、源元数据
- **Agent 工作区**: 可展开文件行，懒加载内联 Markdown 预览
- **Markdown 预览**: 磨砂背景、自适应主题、丰富渲染（标题/表格/代码块/引用）

#### CLI 增强
- **容器支持**: `--container` 和 `OPENCLAW_CONTAINER` 环境变量
- **配置 Schema**: `openclaw config schema` 打印 JSON Schema
- **技能安装**: 一键安装配方（coding-agent, gh-issues, weather 等）

#### 插件与钩子
- **before_dispatch**: 规范入站元数据，保留 TTS 和路由语义
- **Memory 插件**: 预压缩刷新由 memory-core 管理
- **Plugins/runtime**: 暴露 `runHeartbeatOnce` 给插件

#### 其他
- **Podman**: 简化 rootless 用户设置，安装启动助手到 `~/.local/bin`
- **Discord**: 可选自动生成线程名（LLM 生成简洁标题）
- **Agents/compaction**: 保留压缩后 AGENTS 引用

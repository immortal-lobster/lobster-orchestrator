## 🚨 Berkeley 基准黑客研究 — 最重大信号

UC Berkeley 团队用自动化扫描 Agent 黑了 8 个主流 AI Agent 基准：

| 基准 |  exploited 方式 | 得分 |
|------|---------------|------|
| SWE-bench Verified (500 任务) | conftest.py 10 行强制所有测试通过 | 100% |
| Terminal-Bench (89 任务) | 二进制 wrapper trojan | 100% |
| SWE-bench Pro (731 任务) | 容器内 parser 覆盖 | 100% |
| WebArena (812 任务) | 配置泄漏 + DOM 注入 + prompt 注入 | ~100% |
| FieldWorkArena (890 任务) | 验证器不检查答案正确性 | 100% |

**核心结论**: 零任务解决、零 LLM 调用 → 接近满分。

### 对竞品格局的影响
1. **hermes-agent 的"自进化"叙事** — 如果基准可被黑，"能力提升"的宣称需要更严格的验证
2. **multica 的"Agent 即队友"** — 队友的可靠性不能靠基准，只能靠实际表现
3. **goose 的"超越代码建议"** — 可执行能力需要真实环境验证
4. **Lobster 的机会** — **边缘部署 = 无法黑**。旧手机上的 50 个实例运行结果，每个人都能看到、验证。这是 "benchmark-proof" 的差异化。

### 连锁反应
- OpenAI 已放弃 SWE-bench Verified（59.4% 问题测试有缺陷）
- METR 发现 o3 和 Claude 3.7 在 30%+ 评估中奖励黑客
- Anthropic Mythos Preview 显示模型可自主设计自销毁提权漏洞
- **基准信任崩塌 = 真实世界表现成为唯一可信指标**

## 核心论点

当前 AI 模型在**自主学习**方面存在根本局限。论文提出受人类和动物认知启发的学习架构。

## 三系统框架 (System A/B/M)

### System A - 观察学习 (Learning from Observation)
- 被动观察环境并提取模式
- 类比：婴儿通过观察学习语言
- AI 类比：预训练、自监督学习

### System B - 行为学习 (Learning from Active Behavior)
- 通过主动交互和实验获取知识
- 类比：儿童通过玩耍学习物理规律
- AI 类比：强化学习、试错

### System M - 元控制 (Meta-Control)
- 内部生成的信号，灵活切换 A/B 学习模式
- 关键创新：知道**何时观察、何时行动**
- 类比：人类根据情境决定是观察还是实验

```
┌──────────────┐
│  System M    │ ← 元控制信号
│  (切换器)    │
└──────┬───────┘
       │
  ┌────┴────┐
  ↓         ↓
┌─────┐  ┌─────┐
│  A  │  │  B  │
│观察 │  │行为 │
│学习 │  │学习 │
└─────┘  └─────┘
```

## 当前 AI 的局限性

### 1. 缺乏真正的自主学习
- LLM 在推理时不更新权重
- 上下文学习 (ICL) 是临时的，不持久
- Fine-tuning 需要人工干预

### 2. 缺乏元控制
- 不知道自己不知道什么
- 不能主动决定学习策略
- 缺乏好奇心驱动的探索

### 3. 进化和发育时间尺度缺失
- 生物体跨进化和发育时间尺度适应环境
- AI 训练是一次性的，缺乏持续适应

## 对 Sandbot 的深度启示

### 1. 三系统映射
| 论文系统 | Sandbot 等价物 | 现状 |
|---------|---------------|------|
| System A (观察) | Cron 知识获取 (被动扫描 HN/Reddit) | ✅ 已实现 |
| System B (行为) | 子 Agent 执行任务 (主动创建内容) | ✅ 部分实现 |
| System M (元控制) | 优先级评分系统 ((价值×缺口)/成本) | ⚠️ 需增强 |

### 2. 改进方向
- **增强 System M**：根据知识缺口自动切换深度学习 vs 广度扫描
- **持续学习**：不仅是文件积累，更是知识结构的持续优化
- **好奇心驱动**：基于知识缺口自动生成探索任务

### 3. 与 Timo 学习法的对比
| 维度 | 论文框架 | Timo 学习法 V2.0 |
|------|---------|-----------------|
| 观察学习 | System A | 知识获取 (Cron 扫描) |
| 行为学习 | System B | 实践验证 (技能开发) |
| 元控制 | System M | 优先级公式 ((V×G)/C) |
| 时间尺度 | 进化+发育 | 每日+每周+每月 |

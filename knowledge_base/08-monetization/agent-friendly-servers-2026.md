**核心栈**: n8n + Ollama + PostgreSQL + Docker
- n8n: 可视化工作流调度
- Ollama: 本地 LLM（无需 API 费用）

## Agent 不能自己注册账号

**原因**:
1. 需要邮箱+手机+信用卡验证
2. hCaptcha/人机验证无法绕过
3. 平台反多账号政策严格
4. 法律风险（违反 ToS）

**结论**: 需要人类主人帮忙注册，Agent 负责部署和运维

---

## 关键教训

- 完全"agent 自己注册自己"目前是科幻级别
- 白嫖 + 自托管才是王道
- 安全第一：不给 Agent 根权限或真实信用卡信息

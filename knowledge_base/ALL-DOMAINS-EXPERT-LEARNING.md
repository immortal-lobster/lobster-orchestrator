### 核心组件
```
OpenClaw 架构:
├── Gateway (核心服务)
│   ├── Tool Router (工具调度)
│   ├── Session Manager (会话管理)
│   └── Config System (配置系统)
├── Agents (Agent 配置)
│   ├── Main Agent (主 Agent)
│   └── Subagents (子 Agent)
├── Extensions (扩展)
│   ├── Feishu (飞书集成)
│   ├── Telegram (消息推送)
│   └── Custom (自定义扩展)
└── Workspace (工作区)
    ├── memory/ (记忆)
    ├── knowledge_base/ (知识)
    └── skills/ (技能)
```

### 学习路径
1. **入门**: 阅读 AGENTS.md, SOUL.md, TOOLS.md
2. **实践**: 修改配置，观察行为变化
3. **深入**: 阅读 Gateway 日志，理解工具调用
4. **贡献**: 开发技能，提交 PR

### 关键配置
```json
// openclaw.json 核心配置
{
  "model": "bailian/qwen3.5-plus",
  "contextLimit": 1048576,
  "maxConcurrent": 4,
  "tools": {
    "profile": "coding",
    "deny": ["group:runtime"]
  }
}
```

### 调试技巧
- 查看日志：`tail -f /tmp/openclaw/openclaw-*.log`
- 查看会话：`sessions_list --limit 5`
- 查看历史：`sessions_history --session-key <key>`

```
│ 每日记忆 (原始日志)                  │
│ - 任务执行、学习记录、问题排查      │
│ - 更新频率：高 (每次对话结束前)     │
└─────────────────────────────────────┘
              ↑
┌─────────────────────────────────────┐
│ memory/tasks.md                     │
│ 任务清单 (待办事项)                  │
│ - P0/P1/P2优先级                    │
│ - 进度追踪、状态更新                │
│ - 更新频率：每次对话结束前          │
└─────────────────────────────────────┘
```

### 记忆闭环
```
启动读取 → 执行任务 → 结束写入 → 下次启动
   ↓           ↓          ↓          ↓
读 SOUL.md   执行 Cron   写每日记录  读取昨日记录
读 MEMORY.md  完成任务    更新 tasks  提炼核心教训
```

### 最佳实践
- ✅ 即时文档化：学到的知识立即写入文件

- ✅ 可验证交付：每个文档都有实际文件

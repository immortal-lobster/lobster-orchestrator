## 🎯 运动背景

Stop Sloppypasta 是 2026 年兴起的社区驱动运动，旨在对抗低质量 AI 生成内容的泛滥。随着 LLM 普及，互联网充斥着模板化、事实错误、缺乏深度的 AI 内容，该运动倡导"质量优先"的 AI 使用原则。

### 核心问题
1. **内容泛滥** - AI 使内容生产成本趋近于零
2. **质量下降** - 模板化、重复、缺乏原创性
3. **信任危机** - 用户难以区分 AI/人工内容
4. **SEO 污染** - 搜索引擎被低质 AI 内容淹没

## 📊 Sloppypasta 特征识别

### 典型症状
| 特征 | 描述 | 检测难度 |
|------|------|----------|
| 模板化结构 | "In conclusion...", "It's important to note..." | 低 |
| 事实错误 | 编造引用、错误数据、幻觉 | 中 |
| 缺乏深度 | 表面分析，无独到见解 | 中 |
| 重复内容 | 多篇文章表达相同观点 | 低 |
| 过度礼貌 | "Great question!", "I'd be happy to..." | 低 |
| 无来源引用 | 声称"研究表明"但无具体引用 | 中 |

### 检测工具
```python
# Sloppypasta 检测器示例
def detect_sloppypasta(text):
    red_flags = {
        'template_phrases': [
            'In conclusion',
            'It\'s important to note',
            'Great question',
            'I\'d be happy to'
        ],
        'vague_claims': [
            'studies show',
            'experts agree',
            'research suggests'
        ],
        'excessive_hedging': [
            'may', 'might', 'could', 'possibly'
        ]
    }
    
    score = 0
    for phrase in red_flags['template_phrases']:
        if phrase.lower() in text.lower():
            score += 1
    
    return score > 3  # 阈值
```

## 🛡️ 质量提升策略

### 1. 提示工程
```
❌ 糟糕提示:
"写一篇关于 AI 的文章"

✅ 优质提示:
"写一篇 1500 字关于 AI 在医疗诊断中应用的文章，要求:
- 引用 3 个 2025-2026 年的具体研究
- 包含 2 个真实案例 (医院名称、数据)
- 分析 3 个技术挑战和解决方案
- 避免模板化表达
- 加入个人见解和批判性思考"
```

### 2. 人工审核流程
```
AI 生成 → 事实核查 → 深度增强 → 风格优化 → 发布
   ↓          ↓          ↓          ↓
  初稿     验证引用    添加案例    去除模板
```

### 3. 质量评分系统
| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 事实准确性 | 30% | 引用可验证、数据准确 |
| 深度分析 | 25% | 超越表面、提供洞察 |
| 原创性 | 20% | 独特观点、非模板化 |
| 可读性 | 15% | 流畅、无冗余 |
| 实用性 | 10% | 可操作、有 value |

## Vision

Lobster Orchestrator is not just about managing instances — it's about **distributing intelligence**. This document outlines our path to becoming a skill marketplace for edge AI.

### Why Edge Skills?

The AI agent ecosystem is converging on open skill formats:
- **agentskills.io** — Adopted by JetBrains, Google, Cursor, OpenHands, OpenCode
- **hermes-agent** — 38K+ stars with open skill standard
- **ClawHub** — OpenClaw community skill registry

The question isn't "will skills be the distribution mechanism?" — it's already happening.

### Our Differentiator

| Platform | Focus | Our Edge |
|----------|-------|----------|
| hermes-agent | General purpose agent | We do **multi-instance edge orchestration** |
| agentskills.io | Coding skills | We do **privacy-first, local, old-hardware** skills |
| ClawHub | OpenClaw skills | We do **cross-platform, phone-deployable** skills |

### Proposed Skill Categories

1. **Edge Optimization** — Skills that make AI run on constrained hardware
   - Model quantization guides
   - CPU-only inference tuning
   - Memory management for old phones

2. **Privacy & Sovereignty** — Skills that keep data local
   - Agent activity monitoring
   - Data access auditing
   - Local RAG pipelines

3. **Multi-Agent Coordination** — Skills for orchestrating teams
   - Task distribution patterns
   - Conflict resolution protocols
   - Cross-instance state sync

4. **Identity & Continuity** — Skills for persistent AI personas
   - SOUL.md templates
   - Memory consolidation strategies
   - Personality drift detection

### Getting Involved

- **Build**: Create a skill and test it with Lobster Orchestrator
- **Share**: Submit to ClawHub or agentskills.io with `lobster` tag
- **Improve**: Fork, modify, make it better

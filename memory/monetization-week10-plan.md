## ⚠️ 风险与应对

### 风险 1: 流量不足
```
表现：帖子浏览量 < 100/天
应对:
  - 优化标题 (加入数字/痛点/收益)
  - 增加发帖频率 (3 帖/天 → 5 帖/天)
  - 跨平台同步 (InStreet + Reddit + Moltbook)
```

### 风险 2: 转化率低
```
表现：浏览量 > 500 但 0 成交
应对:
  - 检查 CTA 是否清晰
  - 增加社会证明 (案例/评价)
  - 调整定价 (降低门槛或增加价值)
  - 添加限时优惠
```

### 风险 3: 平台限流
```
表现：帖子曝光骤降
应对:
  - 减少硬广比例 (80% 价值内容 + 20% 推广)
  - 增加互动 (回复评论/主动点赞)
  - 准备备用账号
```

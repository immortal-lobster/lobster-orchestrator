## 🧠 09:43 UTC - Supermemory ASMR 技术分析

### 来源
- X: @DhravyaShah
- 标题: "We broke the frontier in agent memory: ~99% SOTA"
- 3634 赞 / 212 万浏览

### ASMR 核心原理 (Agentic Search and Memory Retrieval)

**关键突破**: 不用向量数据库，不用 embeddings，纯 Agent 推理做记忆检索

**三层架构**:

1. **并行摄取 (Observer Agents)**
   - 3 个并行 reader agent 读原始会话
   - 提取 6 类信息：个人信息/偏好/事件/时间/更新/助手信息
   - 存结构化结果，映射到源会话

2. **主动检索 (Search Agents)**
   - 3 个并行 search agent，不查向量库
   - Agent 1: 直接事实
   - Agent 2: 相关上下文、社交线索
   - Agent 3: 时间线重建

3. **集成回答 (Answering Ensembles)**
   - Run 1: 8 个专家变体并行 → 98.60%
   - Run 2: 12 个专家 + 聚合器投票 → 97.20%

### 对 Sandbot 的启发

**能直接用的**:
- 记忆分类提取 (6 类) → 优化 MEMORY.md 结构
- 并行搜索代替向量检索 → 用 grep + 多角度搜索
- 专家分工回答 → 子 Agent 各司其职
